# MyStudy - Репозиторий учебных материалов

Полный набор лабораторных работ, практических заданий и решений по различным дисциплинам, изучаемым в ВУЗе.

**Важно**: Веб-лабораторные работы (web1-web4) подключены как git submodules. Для правильного клонирования используйте:
```bash
git clone --recursive https://github.com/Antichnoste/MyStudy.git
```

## 📚 Структура репозитория

### **1 Информатика**
- **Лаба 1**: Базовые концепции информатики
- **Лаба 2**: C++ проекты (`laba2.cpp`)
- **Лаба 4**: Работа с XML и JSON форматами на Python
  - XML, JSON, CSV обработка
  - Парсинг и трансформация данных
  - Несколько вариантов решения с измерением производительности
- **Лаба 5**: Анализ финансовых данных на Python
  - Работа с CSV файлами (`SPFB.RTS-12.18_180901_181231.csv`)
  - Использование pandas для анализа временных рядов
  - Тестирование (`test.py`)
- **Лаба 6**: LaTeX документы
  - Составление технических документов
  - Работа с шаблонами и изображениями

### **1-2 Основы профессиональной деятельности (ОПД)**
Курс по архитектуре ЭВМ и программированию на ассемблере для платформы Helios

- **Lab1-Lab2**: Базовые операции и команды
- **Lab3**: Промежуточный уровень
- **Lab4**: Сложные структуры
- **Lab5-Lab7**: Ассемблер программы для платформы Helios
  - Мониторинг и управление процессами
  - Анализ использования CPU и памяти (скрипты на bash)
  - Работа с топ-процессами

---

### **1-2 Программирование**
Лабораторные работы по объектно-ориентированному программированию на Java

- **Lab2**: Система для боевых действий покемонов
  - Классы `Pinsir`, `Feebas`, `Clefable`, `Clefairy`, `Cleffa`
  - Система боя и управления параметрами
  - Внешние библиотеки для боевых симуляций

- **Lab3**: Продвинутые концепции ООП
  - Records, Enums
  - Исключения и обработка ошибок
  - Action, Adventure паттерны

- **Lab5**: Консольное приложение с системой управления
  - Менеджеры и утилиты
  - Скриптовая система (script1, script2, script3)
  - Работа с командами и внешними библиотеками

- **Lab6**: Клиент-серверное приложение
  - Архитектура: client/server/common
  - Использование log4j2 для логирования
  - Работа с ресурсами и конфигурацией

---

### **2 Базы данных**
Лабораторные работы по проектированию и работе с реляционными БД

- **Lab1**: Введение в БД
  - Создание таблиц с использованием типов и доменов
  - Вставка данных
  - Базовые запросы
  - Примеры: таблицы WORLD, LEVEL, BODY (для проекта на тему "Божественная комедия")

- **Lab2**: Сложные запросы
  - 7 различных SQL запросов разной сложности
  - Использование JOIN, агрегирующих функций

- **Lab3**: Нормализация и целостность БД
  - Нормализация схемы
  - Триггеры для поддержания целостности
  - Денормализованные таблицы

- **Lab4**: Продвинутые запросы
  - Оптимизация запросов
  - Работа с индексами
