# Каркас для блока 2

## Файлы
- `objective.py` - целевая функция, градиент, Гессиан и параметры задачи
- `model/base.py` - базовые абстракции оптимизаторов и модели результатов
- `model/methods.py` - классы численных методов
- `model/analytics.py` - аналитические стационарные точки и их классификация
- `model/visualization.py` - построение линий уровня и траектории итераций
- `model/report.py` - сборка и сохранение отчета
- `main.py` - единая точка запуска всех частей задания

## Запуск
```bash
python main.py
```

## Что формируется
- Три графика для ДЗ2.1 в папке `outputs`
- Отчет `outputs/block_2_report.txt` на русском языке
